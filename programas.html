Justificación

El beneficio a obtener por parte del aplicativo será generar un control de accesos con el fin de que el usuario pueda realizar su loggin con sus datos registrados por el área de administración para que puedan realizar un loggin además de tener factores de recuperación en caso de pérdida de contraseña

Objetivo

Diseñar, y desarrollar una aplicación con base de datos, para dar solución a un problema específico de administración y seguridad.

Conclusiones

En base a todo lo expuesto anteriormente podemos concluir que es de gran importancia la implementación de este sistema, el cual proveerá a la Universidad grandes beneficios tales como el Sistema de ingreso a los estudiantes, la facilidad en la búsqueda de datos de sus usuarios y la organización de su base de datos entre otras; prestando así un mejor servicio. Además, podemos decir que este proyecto ha sido de mucho provecho para nosotros debido a que hemos podido entender el funcionamiento de las diferentes fases del ciclo de vida
